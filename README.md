So I will be working on [Libreoffice Themes]() this summer, where I will be adding support to customize the
UI, implementing "Themes" in libreoffice. Libreoffice has it's own GUI Library VCL, which interfaces with
different widget toolkits like GTK, QT, Win32 etc to create dialogs etc.

Current state is that VCL just extracts colors out of the widget toolkits, but doesn't seem to put back
the updated colors. That's what I will be doing.
